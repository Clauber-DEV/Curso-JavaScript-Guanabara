operadores aritiméticos e de atribuição:

> 9 % 2 == 1
> 9 / 2 == 4.5
> 5 + 3 / 2 == 6.5
> (5 + 3) / 2 == 4

Ordem de precedência:

> 1º ( )

> 2º **
> 3º * / %
> 4º + -

Auto atribuição:

var n = 3

n = n + 4 == n += 4
n = n - 5 == n -= 5
n = n * 4 == n *= 4
n = n / 2 == n /= 2
n = n ** 2 == n **= 2
n = n % 5 == n %= 5
