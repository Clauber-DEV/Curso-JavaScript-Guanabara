Operadores de identidade:

o sinal == compara se é igual

o sinal != compara se é diferente

o sinal === compara se é identico

o sinal !== compara se é desigual restrito

5 == 5 true

5 == '5' true

5 === '5' false

5 === 5 true

! negação

se for falso se torna verdade
se for vardade se torna falso

&& conjunção

|| disjunção
